What is Forza Horizon 4?

Forza Horizon 4 is a racing video game set in an open world environment based in a fictionalized Great Britain, with regions that include representations of Edinburgh, the Lake District (including Derwentwater), Ambleside, and the Cotswolds (including Broadway), among others, and features currently over 700 licensed cars. The game features a route creator that allows players to create their own races. The game takes place in a shared world, compared to the AI-driven 'drivatars', with each server supporting up to 72 players. The game is also playable in offline mode. Players have the opportunity to buy in-game houses that unlock new items, cars, and game-play perks. The game features a weather system that also changes the seasons. The environment in the world will change depending on the season: for example, Derwentwater would freeze over in winter and allow players the ability to drive on the ice to reach areas of the game world that would be inaccessible during all the other seasons. The seasons are fixed across the game, meaning that all players will experience the same conditions at the same time. After completing a series of events that introduce players to all four seasons, the shared-world seasons will change every week.
